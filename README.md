# AgentFlow VSCode Extension

Browse and install workflows directly from VSCode.

## Features

- Browse workflow marketplace
- Install workflows with one click
- Quick command palette access

## Commands

- `AgentFlow: Browse Workflows` - Browse the marketplace
- `AgentFlow: Install Workflow` - Install a workflow

## Installation

```bash
code --install-extension agentflow-vscode
```

## Usage

1. Open Command Palette (Cmd/Ctrl + Shift + P)
2. Type "AgentFlow" to see available commands
3. Select a workflow to install
